PROBLEM STATEMENT:

Write a python program which will ask for 3 integers. These 3
integers are subjects marks of a student. 3 subjects like Math, Physics,
Chemistry. You have to find the merit from these 3 integers. 
The average of these 3 marks is the merit.

Sample Input 1:
67
45
78
Sample Output 1:
63.333

Sample Input 2:
89
90
89
Sample Output 2:
89.333

Sample Input 3:
90
90
90
Sample Output 3:
90

Constraints:
0 <= marks <= 100
